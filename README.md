# Dante Martinez · Landing Page

Personal landing page / portfolio for **Dante Martinez** — AI Solutions Architect & Automation Engineer based in Monterrey, MX.

Built as a single static `index.html` so it can be served directly from **GitHub Pages** with no build step.

## Stack
- Plain HTML + CSS (inline) + a few lines of JS
- Google Fonts (Lora + DM Sans), lazy-loaded
- Inline lucide-style SVG icons (no icon library bundle)

## Aesthetic
Follows the **Nostalgia Sistemas** spec — editorial, warm, noreste-de-México feel. Cream + carbon section rhythm, ambar/rosa accents, Lora serif heads with italic emphasis lines.

## Local preview
Just open `index.html` in a browser. No build, no dependencies.

## Deploy (GitHub Pages)
1. Push to `main`.
2. Repo → Settings → Pages → Source: `main` / root.
3. Done — site serves `index.html` from the repo root.
